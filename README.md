# 家园守望塔防

静态 HTML5 Canvas 塔防游戏，可直接部署到 GitHub Pages、Netlify、Vercel 或任意静态文件服务器。

## 在线访问

- GitHub Pages：<https://auroraastra.github.io/tower-defense/>

## 当前版本

- 版本：0.9.0
- 关卡：20 关
- 防御塔：5 种，按通关进度逐步解锁
- 波次：每关 5 波，自动倒计时刷波
- 怪物成长：血量、移速、体积随关卡上升
- 特殊怪物：高速、护盾、护甲、回血、Boss；Boss 会鼓舞小怪并震慑附近防御塔
- 无尽模式：所有防御塔直接可用，使用更多专用建造点，每 5 波一个 Boss，怪物强度随波次持续上升
- 无尽升级：防御塔没有等级上限，升级费用和输出成长同步递增
- 存档：使用浏览器本地存储保存已解锁关卡和最近选择关卡
- 玩家档案：可用昵称注册/切换本机玩家档案，不同玩家独立保存进度、星章和商店升级
- 存档导入导出：选关页可导出全部玩家档案，也可导入 JSON 存档
- 流程：先进入选关界面，再进入战斗
- 战前准备：每关开始前展示游戏规则、挑战任务和可选战术卡
- 战前情报：开局前汇总本关特殊怪词条，并给出推荐防御塔
- 挑战任务：无伤、清障、限塔数、雪花塔控制等目标会在战斗中追踪，完成后奖励金币
- 星章商店：挑战任务额外奖励星章，可购买永久开局金币、房屋耐久、雪花塔专精
- 战术卡：霜线预案、清障合同、抢攻节拍提供不同开局策略
- 局内成长：战役每关最多触发 2 次三选一 Buff，无尽每 3 波触发一次；清理水晶可提供一次早期成长机会
- 剧情浮层：关卡接入、Boss 入侵、漏怪、失败、通关等事件会弹出 NEURO-GRID 终端提示
- 信息面板：选中防御塔后展示伤害、射程、溅射、穿透、破甲、破盾和减速；HUD 展示当前 Buff 叠层
- 随机召唤：选择空建造点后可消耗金币随机获得已解锁基础塔
- 合成升级：选择防御塔后，可将同类型同等级的另一座塔合成为随机高阶塔
- 撤退再部署：选择防御塔后可撤退回收金币，再根据战局重新布阵
- 音效：使用 WebAudio 合成建造、开波、命中、奖励、Boss 和胜负反馈音，可在战斗 HUD 中开关
- 调试入口：战斗 HUD 提供调试面板，可加金币、通关、解锁关卡或召唤 Boss
- 交互：点击建造点选择可用防御塔；可升级塔会在塔上方显示升级标识
- 移动端：优化手机横屏布局，战斗 HUD 提供完整底部标签切换状态、关卡、任务、塔、Buff 和操作
- Playwright：包含基础端到端测试文件，覆盖选关、移动底部标签、调试入口和战斗启动

## 文件

- `index.html`：页面结构
- `styles.css`：HUD 与响应式样式
- `config.js`：版本号、塔、关卡、敌人、Buff、商店等配置与关卡生成
- `game.js`：游戏状态、存档、战斗逻辑、UI 交互和 Canvas 渲染

## 后续扩展位置

- 新塔：修改 `config.js` 中的 `towerTypes` 与 `towerOrder`
- 关卡路径：修改 `config.js` 中的 `pathSets`
- 建塔点：修改 `config.js` 中的 `buildSets`
- 地图道具：修改 `config.js` 中的 `propSets`
- 战役怪物成长：修改 `config.js` 中的 `createWave`
- 无尽怪物成长：修改 `config.js` 中的 `createEndlessWave`
- 初始资源与关卡生成：修改 `config.js` 中的 `createLevels`
- 局内 Buff：修改 `config.js` 中的 `draftCards` 和 `game.js` 中的 `towerStats`
- 随机召唤费用：修改 `randomSummonCost`
- 合成规则：修改 `mergeSelectedTower`
- 自动刷波间隔：修改 `AUTO_WAVE_DELAY`

## 本地运行

```bash
python3 -m http.server 8127 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:8127/
```

直接打开 `index.html` 也能运行，但部署前建议使用本地服务器测试。
